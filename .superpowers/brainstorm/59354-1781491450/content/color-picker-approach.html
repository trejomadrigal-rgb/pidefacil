B — Color libre (hex picker)

El dueño escribe cualquier color hex o usa un selector. Máxima flexibilidad para que coincida con su marca real. Requiere validar contraste mínimo.

En ambos casos el color se guarda en Business.menuColor y se pasa como CSS variable --brand al menú público. El header oscuro (#1A1A2E) siempre permanece igual — solo cambia el color de acento.
